# boston_housing

## Installation

Suivre les indications ci-dessous :

```bash
git clone https://github.com/Simplon-Martin/boston_housing.git
cd boston_housing
```

Sous Windows : 

```bash
python -m venv venv
```

Sous Linux : 

```bash
source venv/bin/activate
```

Ensuite : 

```bash
pip install -r requirements.txt
```
